# 数据分析

数据分析的基础是数据的收集、处理和分析。

数据的质量往往决定了最后分析结果的质量，
这个网站的目的打算逐步收集一些公开的数据并共享出来，供学习数据分析之用。

## 数据使用

每个菜单最后提供了数据下载地址。

数据都是 `csv` 格式的，编码是`UTF-8`。

windows 系统上直接用 excel 打开可能会乱码，建议用文本编辑器打开查看。

或者直接用 python 的 pandas 库来打开：

```python
import pandas as pd

df = pd.read_csv("xxxx.csv")
df
```
